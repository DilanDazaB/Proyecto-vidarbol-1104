---

## 🎯 Objetivo del Proyecto

Nuestros propósitos generales son:

- Aprender a plantar y cuidar un árbol.  
- Enseñar a las personas lo divertido que es proteger el ambiente.  
- Concientizar a la comunidad sobre la importancia de los árboles para la vida.
